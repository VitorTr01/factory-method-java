# 📌 Factory Method em Java

Projeto com implementação do padrão Factory Method, feito durante meus estudos para entender melhor na prática como funciona a criação de objetos de forma organizada.

## 💻 O que tem aqui
- Aplicação do padrão Factory Method
- Organização de código
- Separação de responsabilidades
- Exemplo prático

## 🛠️ Tecnologias
- Java

## 🎯 Objetivo
Entender na prática como usar padrões de projeto para melhorar a estrutura do código.

## 📈 Sobre
Projeto voltado para prática de Design Patterns, mostrando uma forma mais organizada e escalável de desenvolver soluções.
